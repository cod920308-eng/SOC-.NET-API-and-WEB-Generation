# SOC .NET API and WEB Generation

## Overview
Service Oriented Architecture project built with .NET technology, consisting of a REST API and web applications.

## Project Structure
SOC-.NET-API-and-WEB-Generation/
├── KmcAPI/          # .NET Web API
├── Web1/            # Aloka Event Organizers
└── Web2/            # Kandy Municipal Council - Events web

## Technologies Used
- ASP.NET Core Web API
- Entity Framework Core
- AutoMapper
- SQL Server
- CORS enabled

## Getting Started

### Prerequisites
- .NET 8 SDK
- SQL Server
- Visual Studio 2022+

### Run the API
cd KmcAPI
dotnet restore
dotnet run

## API Endpoints
- Events
- Organizers
- Participants
- Event Registrations
- Admin
